# Smartphone-Tethered-Visual-Field-Mapping-and-Neuro-Rehabilitation-Wearable.
A hardware and software system designed to assist in the recovery of visual field deficits (such as those caused by stroke or traumatic brain injury).The system consists of:

1. Hardware: A wearable headset/glasses equipped with an internal display and high-speed infrared cameras for real-time eye-tracking and gaze-contingent stimulus delivery.

2. Software/Compute: A mobile application (running on a smartphone connected via USB/Bluetooth) that handles the computational load for gaze tracking, stimulus generation, and data logging.

3. Functionality: The system maps areas of visual loss (scotomas) by tracking user response to displayed stimuli. It utilizes gamification and repetitive task-based neurological training to encourage visual search and potential neuroplastic recovery in affected visual sectors.

Status: Conceptual design and software architecture established prior to the commencement of employment. Current development involves prototype iteration and eye-tracking algorithm refinement.
